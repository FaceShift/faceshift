# FaceShift
An electron app that uses facial tracking to move your mouse cursor.

## Pre-requisites 

* Node + npm

* Electron 
```
$ npm install -g electron
```

## Run the project
1. Clone the project
```
$ git clone https://github.com/FaceShift/faceshift.git
```
2. Run the project
```
$ cd faceshift
$ npm install
$ npm start
```
